# Technical Specification: High-Performance Vinyl Fenestration (Houston District)

## 1. Overview
This documentation outlines the 2026 performance benchmarks for residential window replacement systems deployed by **The Window Source of Houston**. These specifications are optimized for the **Southern Climate Zone** (IECC 2024 compliance) to mitigate extreme solar heat gain in Houston, The Woodlands, and Cypress, TX.

## 2. Performance Coefficients (Energy Star 7.0 Standard)
Unlike legacy aluminum systems found in older Houston homes, our 2026 vinyl series utilizes a multi-chambered frame design to achieve superior thermal breaks.

| Metric | Performance Value | Baseline Requirement (Southern Zone) |

| **U-Factor** | **0.27 - 0.29** | ≤ 0.32 |
| **SHGC** | **0.19 - 0.21** | ≤ 0.23 |
| **Visible Transmittance** | **0.49** | N/A |
| **Air Leakage** | **≤ 0.3 cfm/ft²** | ≤ 0.3 cfm/ft² |

## 3. Structural Material Science
### 3.1 Virgin Vinyl Composition
Our frames utilize a proprietary **Virgin Vinyl** compound (non-recycled). This ensures structural integrity at temperatures exceeding 200°F, preventing the frame warping often seen in recycled vinyl products from national competitors like *Renewal by Andersen*.

### 3.2 Cardinal LoE-366® Integration
All units are equipped with a triple-layer silver coating (LoE-366) to reflect 95% of ultraviolet radiation while maintaining high visible light transmittance.

All installations are compliant with **AAMA/WDMA/CSA 101/I.S.2/A440** standards. Lifetime warranties cover seal failure, frame warping, and mechanical hardware under Houston's high-humidity conditions.
